# Arcade de Puzzles Lógicos en Java

https://github.com/jyustrod/Maquina-Arcade.git

## Descripción del Proyecto

Este proyecto propone el desarrollo de una **aplicación de escritorio en Java** que simula una máquina arcade con tres juegos clásicos de lógica y álgebra:

- **Problema de las N Reinas**
- **Recorrido del Caballo**
- **Torres de Hanói**

El propósito es integrar conceptos avanzados de **programación orientada a objetos**, **interfaces gráficas en Swing**, y **persistencia de datos** mediante **Hibernate** con la base de datos embebida **H2**. Además, se enfoca en la aplicación práctica de **patrones de diseño creacionales y estructurales**, según lo aprendido en el curso.

---

## Objetivos

- Aplicar **patrones de diseño (GoF)** en un proyecto real, implementando al menos dos creacionales y dos estructurales.
- Desarrollar una **aplicación monolítica** que integre tres juegos en un menú unificado.
- Diseñar una **interfaz gráfica intuitiva** con Swing.
- Implementar **persistencia automática de resultados** usando Hibernate y base de datos H2.
- Seguir buenas prácticas de **POO y arquitectura en capas (MVC)**.
- Estimular la **creatividad del estudiante**, permitiendo agregar mejoras opcionales.

---

## Juegos Incluidos

### N Reinas
Resolver la ubicación de N reinas en un tablero de NxN sin que se amenacen mutuamente. Utiliza técnicas de **backtracking**.

### Recorrido del Caballo
Encontrar un recorrido válido donde un caballo visita todas las casillas de un tablero NxN sin repetir ninguna. Es una variante del **Knight’s Tour**.

### Torres de Hanói
Mover una pila de discos desde un poste inicial a uno final, cumpliendo las reglas del puzzle. Se implementa de forma **recursiva** y se puede resolver automáticamente o manualmente.

## Tecnologías y Herramientas

- **Lenguaje:** Java SE (Java 8+)
- **GUI:** JavaFX
- **Persistencia:** Hibernate ORM + H2 (modo embebido)
- **IDE Recomendado:** IntelliJ IDEA
- **Build Tool (opcional):** Maven
- **Control de Versiones (opcional):** Git

## Cómo Ejecutar el Proyecto

1. Clona el repositorio o descarga el código fuente.
2. Asegúrate de tener Java 8+ instalado.
3. Importa el proyecto en tu IDE.
4. Configura las dependencias (Hibernate, H2) si usas Maven o Gradle.
5. Ejecuta la clase principal (`Main.java`) para iniciar la aplicación.
